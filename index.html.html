Unity Web Player | Nightmares

This game was created by Tiff Zhang via Unity's Survival Shooter Tutorial.

It was done as an assignment for Georgia Tech's game design class CS 4455, and the following significant edits were made to the game:

  • Ammo is limited. There is a slider to display ammo and there are specific ammo spawn points on the map. In addition, enemies may drop ammo. To balance this change, the gun damage was significantly increased.
  • The level is twice the size of the original tutorial level, and more objects were added to fill up this extra space. Additionally, there is an extra set of spawn poitns for all of the monsters.

The source code for this project is available on Github.
