Unity Web Player | Nightmares

This game is based on Unity's survival shooter tutorial.

It was created for Georgia Tech's video game design class CS 4455. The following changes to the game from the tutorial version were made:

  • Ammo is limited
  • There are ammo spawn locations
  • Monsters have a chance to drop ammo
  • Hellephants have a chance of dropping health
  • The size of the level is twice as big
  • Bullets are significantly stronger
  • Monsters spawn faster
  • There is a cap on the number of monsters that can spawn
